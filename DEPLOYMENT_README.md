# Chess App - Redesign

Clean deployment package for the redesigned Chess App with Coach Jakie Opening Trainer.

## Prerequisites
- Node.js >= 20.0.0
- npm

## Installation
1. Extract this folder
2. Run: npm install
3. Run: npm run dev
4. Open http://localhost:3000

## Build for Production
npm run build
npm start

## Features
- Coach Jakie Opening Trainer with SM-2 spaced repetition
- Interactive Chess Board with legal move validation
- Recall Mode for blind recall training
- Deviation tracking and analysis
- User progress dashboard
- Puzzle training system
